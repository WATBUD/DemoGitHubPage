Edge AI工程師培訓計畫簡介

AI發展至今已愈來愈普及,市場迫切需要更有效率地在應用端(Edge端)部署AI模型,完成服務的落地或產品的量產,而這一切 仰賴Edge AI工程師來實現。

Edge AI工程師需具備的能力包括
1-將訓練好的AI模型進行模型轉換優化(Model Optimization / Model compression ...)
2-將優化模型快速部署到推論執行環境(Inference Runtime),如工廠、醫療、交通、零售等各種應用場域。
3- 開發終端應用程式與推論執行環境(Inference Runtime)協作的能力

正所謂「工欲善其事、必先利其器」,Edge AI工程師需要有強大好用的Edge Al推論佈建及維護工具,Intel的OpenVINO Toolkit 正是其首選。

OpenVINO是一個開源碼跨平台且免費的AI推論開發工具套件,你可使用常見的A訓練框架,例如TensorFlow、Pytorch和Caffe 開發所需的模型,再運用用OpenVINO來轉換和優化後,快速部署實現目標市場的邊緣AI應用;開發人員只需編寫一次演算法,即 可跨各種Intel 運算裝置進行開發及部署。

OpenVINO Runtime 應用介紹

OpenVINO是一個由Intel發起的開源碼跨平台Al推論執行環境開發套件

在模型構建階段,透過Open Model Zoo提供的200多套預訓練模型來進行專案原型的設計,讓你可以更輕鬆地轉換模型並為推論 (Inference)布署做好準備。而且你不用擔心手邊沒有足夠AI運算環境,因為有Intel® DevCloud雲端平台可以讓你在線上體驗完整 的Edge Al推論與佈建歷程。

在優化與布署階段,OpenVION更是讓你如虎添翼的開發工具。它讓你可使用模型優化器來轉換訓練好的模型、運用用Runtime API (推論引擎)來實現深度學習影像辨識與物件辨識,還有Accuracy Checker、Post-Training Optimization、DLWorkbench及加速 部署的好用工具。
